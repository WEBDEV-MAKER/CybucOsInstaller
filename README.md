# CybucOS

### modified debian release

CybucOS is a modified debian release designed to speed up your use of your PC.

## Our features:

* Preinstalled with Steam for gaming
* Preinstalled with FireFox for fast browsing
* Based on debian so you can install most apps
* Preinstalled with wine to run windows apps on linux
* Preinstalled with wineglass to give wine a UI
* Preinstalled with pavucontrol to customize which microphone/speaker source to use
